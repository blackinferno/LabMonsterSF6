## UPDATE 1.0.1 | 2026-02-18 | important | tags: Combo, Import, Validation, Updates
### JP
- 起動時の速度を改善しました。
- 選択していたキャラや操作モードを記憶するようにしました。
- データが自動保存される間隔を調整しました。
- 大規模なインポート等の前に自動的にバックアップを行う機能を追加しました。
- 複数のポイントからデータを復元できる機能を追加しました。
- インポート用の表記辞書を追加し、JP/EN別名の正規化とユーザーが追加できるようにしました。
- XLSXインポートに列マッピング機能を追加し、テンプレート以外の取り込みを強化しました。
- 選択中の操作方法に応じて、自動的に操作タイプが適応されるようになりました。（保存済みのコンボも自動的に反映されます）
- 選択中の操作モードのコンボのみ表示されるようにしました。
- コマンド表記と操作モードが不一致の際に警告が表示されるようにしました。
- Dゲージ効率（ダメージ/Dゲージ消費[負値]）の自動計算機能を追加しました。
- 更新履歴と今後の予定を情報ページに統合し、更新時は差分表示のポップアップを表示するようにしました。
- その他UIの調整（バージョン表記、ヘッダーの高さの調整など）

### EN
- Improved startup speed.
- Made the tool to remember the character and control mode you were using.
- Adjusted the frequency of auto-saves.
- Added a function to automatically backup data before imports and other major changes.
- Added notation dictionary for imporing with JP/EN alias normalization and user-defined entries.
- Added XLSX column mapping import to handle non-standard sheets more reliably.
- Now the selected control mode will be automatically selected based on the mode you are currently using (automatically applied to already saved combos).
- Now only combos for the selected control mode will be displayed.
- Added command validation and control-mode mismatch warnings.
- Added automatic Drive Gauge efficiency calculation (damage/drive gauge spent[nagative value]) based on entered value.
- Unified changelog/roadmap under Information and added version-diff update popup.
- Other UI adjustments (version display, header height adjustment, etc.)
